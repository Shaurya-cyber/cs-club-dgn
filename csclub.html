// About Us

More than just a club

We're a community of students passionate about technology. Whether you're a total beginner or an experienced coder, there's a place for you here. We meet twice a month, both mornings and afternoons, same information attend either session.

📅  Meetings: Mornings: 7:45–8:10AM or Afternoons: 3:25–4:00PM
📍  Location: Room 308
✉️  Any Questions? Email us at csclub.dgn@gmail.com!

See code in action

Coding is learned best through experimentation. Run the program below and see what happens — then tweak it yourself using the link.

main.py 
print("Welcome to the Computer Science Club 💻")
print("Learning. Coding. Creating.\n")

name = input("Enter your name: ")

if name != "":
    print("Welcome to the club,", name + "!")
else:
    print("Welcome, future coder!")

print("\nDive In The World Of Code With Us 🚀")
