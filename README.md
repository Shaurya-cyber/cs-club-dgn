## DGN Computer Science Club — Official Website

The official club website for the **Downers Grove North High School Computer Science Club**. Built and maintained by student officers.

🌐 **Live Site:** [https://github.com/Shaurya-cyber/cs-club-dgn]

---

## About the Club

The DGN CS Club is a student-led organization open to all DGN students — no experience required. We meet twice a month (morning and afternoon sessions, same content) and run workshops, build projects, compete in events, and host guest speakers.

- 📅 **Meetings:** 7:45–8:10 AM or 3:25–4:00 PM, twice a month
- 📍 **Location:** Room 308
- ✉️ **Contact:** csclub.dgn@gmail.com

## Running Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/your-org/dgn-cs-club-website.git
   ```
2. Open `index.html` in any browser — no server or install needed.

---

## Deployment

The site is hosted on **GitHub Pages** from the `main` branch.

Any push to `main` automatically updates the live site within a minute or two. Always test changes locally before merging.

> **Recommended workflow:** make changes on a `dev` branch → test locally → open a pull request → merge to `main`.

---

## Contributing

This repo is maintained by CS Club officers. To suggest a change:

1. Fork the repo or create a new branch off `dev`
2. Make your changes
3. Open a pull request with a short description of what you changed and why
4. An officer will review and merge

Please write clear commit messages — e.g. `add spring hackathon to events` not `update`.
